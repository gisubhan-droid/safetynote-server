#### TASK-003 — 공사요청번호 자동생성 옵션

| 항목 | 내용 |
|------|------|
| **분류** | 기능 추가 |
| **내용** | "요청번호없음" 체크박스 → `LM_YY.MM.DD_##` 자동생성 (KST 기준) |
| **예상 소요** | 1 ~ 2시간 |
| **위험도** | ⭐☆☆☆☆ |

| **상태** | ⏳ 대기 |

**자동번호 형식**
```
LM_YY.MM.DD_##
예시: LM_26.06.19_01 / LM_26.06.19_02
```
- `LM` — 고정값
- `YY.MM.DD` — 등록일 KST 기준 자동
- `##` — 당일 등록 순번 2자리 (01, 02, 03 ...)

**✅ 확정된 삭제 처리 방침**
